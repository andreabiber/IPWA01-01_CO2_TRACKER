# IPWA01-01_CO2_TRACKER
Umsetzung der Aufgabenstellung 1: CO₂-Footprint des Kurses IPWA01-01, der IU internationale Hochschule. 

Name der Non-Profit-Organisation: CO₂ Tracker

# Features
- Einsatz von Bootstrap
- eigenständig gestaltetes Logo und Titel der Non-Profit-Organisation
- Header, globale Navigation, Content-Bereich und Footer mit rechtlichen Hinweisen
- lokale Navigation mit kultureller Anpassung an unterschiedliche Schriftgewohnheiten
- responsive Seite
- Tabelle mit CO₂-Emissionsdaten
    - nach jeder Spalte sortierbar
    - nach Land und Unternehmen filterbar
- Absicherung des Codes gegen potenziell injizierten Code
